## 📋 Descripción

Sistema automatizado de análisis de eventos deportivos usando **Machine Learning Ensemble** para generar picks de apuestas con:

✅ **Validación de fechas** - Solo eventos futuros > 48 horas
✅ **ML Ensemble** - XGBoost, LightGBM, Neural Network
✅ **Cálculo de EV** - Expected Value (ventaja estadística)
✅ **Base de datos** - Histórico completo de picks
✅ **API REST** - Endpoints para análisis automatizado
✅ **Production Ready** - Deployable en Render gratis
